# crop-weed-detection
# 🌱 Crop and Weed Detection using YOLOv8

## 📌 Project Overview

This project focuses on detecting crops and weeds from agricultural images using Artificial Intelligence and Computer Vision.

The model is developed using **YOLOv8**, a deep learning-based object detection algorithm, and implemented using Python in Google Colab.

The main goal is to support smart farming by identifying weeds and crops automatically.

---

## 🚀 Technologies Used

- Python
- YOLOv8
- OpenCV
- Google Colab
- NumPy
- Matplotlib
- Ultralytics

---

## ✨ Features

✅ Detects crops and weeds from images  
✅ Uses YOLOv8 object detection  
✅ Displays bounding boxes with confidence scores  
✅ Supports smart agriculture applications  
✅ Deep Learning based solution
